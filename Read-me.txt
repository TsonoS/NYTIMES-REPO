PROJECT SETUP

1. ONCE THE PROJECT IS EXTRACTED, 
NAVIGATE TO THE FOLDER CONTAINING THE PROJECT USING THE COMMAND LINE 

2. RUN THE FOLLOWING COMMAND : npm install

THIS SHOULD INSTALL THE NODE_MODULES REQUIRED DEPENDENCIES, 
A FOLDER WITH THAT NAME SHOULD APPEAR IN THE PROJECT STRUCTURE.

3. RUN THE FOLLOWING COMMAND : npm start

THIS COMMAND SHOULD AUTOMATICALLY OPEN THE PROJECT ON YOUR PREFERRED BROWSER.
THE PROJECT SHOULD RUN AT THIS URL : localhost:3000

THANK YOU..
